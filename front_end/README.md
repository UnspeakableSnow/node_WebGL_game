# ブラウザFPS「おじさんオフェンス」 - クライアント

- APP
  - login
  - room_access
  - graphic_view
    - deathmatch_data_stream
    - hud_view

room_idの一文字目が&の場合ゲーム以外画面（ロビー、武器閲覧、訓練場）

## コマンド群

```powershell
npm install
npm run serve
npm run build
npm run lint
```
