## 遊び方

0. `docker node npm`を使えるようにする。
1. Dockerを起動する。

```powershell
docker build -t nwg .
docker run -d -p 3000:8080 nwg
```

2. どっかーを起動した鯖のlocalhostなりローカルIPの3000ポートにアクセスする。  
  `localhost:3000`
3. ゲームする。(*ﾉ´∀`)ﾉ
